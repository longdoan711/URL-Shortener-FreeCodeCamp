URL Shortener Microservice

Source code can be found in here

Example:

https://urlshorten-hoanglong7421.herokuapp.com/new/https://google.com

Example Output:

{"origin_url":"https://google.com","short_url":https://urlshorten-hoanglong7421.herokuapp.com/89631}

Example 2:

https://urlshorten-hoanglong7421.herokuapp.com/89631

It will redirect to https://google.com
